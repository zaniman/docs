# GridBase

A decorative frame component that adds visual depth and styling to content frames. Creates corner cutouts and internal shadows/highlights to give frames a 3D beveled appearance.

![GridBase](/images/guide/synthexa-111@2x.png)

## Global Design Changes

GridBase is a global component used throughout the site. To modify its visual design:

1. Click **Edit Component** in the Properties panel
2. You'll see the component's internal structure and design elements
3. Edit the visual properties (corner radius, shadow colors, shadow intensity, highlight effects, etc.)
4. **All instances of GridBase across the entire site will automatically update** with your changes

This ensures consistent visual styling and depth effects throughout your website.

:::warning
GridBase is a foundational decorative component used in nearly every frame on the site. Changes to it affect the visual appearance of nearly every page on your site. Test thoroughly before publishing changes.
:::
